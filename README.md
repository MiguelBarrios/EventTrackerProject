# EventTrackerProject

## Description
This project is a REST endpoint for performing CRUD operations on a database
containing exercise activity information.



## REST API Request's

#### Get event by event id
`GET /exerciseset/{id}`


#### Get all events in database
`GET /exerciseset`


#### Create new event
`POST /exerciseset`

#### Update event
`PUT /exerciseset/{id}`

#### Delete event
`DELETE /exerciseset/{id}`

## Technologies Used
- Spring Boot
- MySQL
- Spring Data JPA
- Gradle
- JUnit
